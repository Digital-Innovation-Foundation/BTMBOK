Main Description

Method Content is fundamentally described by defining Tasks described with Steps that have Work Products as input and output and which are performed by Roles.  Roles also define important responsibility relationships to work products.

The fundamental Method Content abstractions are:

  • Work Product, which includes Artifact, Deliverable and Outcome
  • Role
  • Task
